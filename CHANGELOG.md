# Changelog

## 0.1.3 - 2026-05-17

### Changed

- 資料查詢的退撫表改為「退撫制度對照」，同時呈現舊制、新制與個人專戶制試算欄位，避免誤以為只支援舊制。
- 釐清退撫月扣款呈現：退撫基金制與個人專戶制個人負擔公式目前相同，表格不再列出兩個相同的舊制／新制欄位。
- 薪資試算預設情境改為土木初任公務員：薦任六本一、專業加給表七、工程另增支、個人專戶制。

## 0.1.2 - 2026-05-17

### Added

- 首頁圖改為符合現有暗色配色的公務員日常黑色幽默漫畫風格。
- 新增退撫「個人專戶制（112年7月1日後初任）」選項，依本俸 × 2 × 15% × 35% 試算個人月提繳額。
- 分享圖卡新增考績等級與年終月數設定，年收差額納入月薪差、考績差與年終差。

### Fixed

- 修正 GitHub Pages 子路徑部署時首頁圖使用絕對路徑導致遺失的問題。
- 分享圖卡下載改為輸出圖卡本體透明背景，移除突兀外框背景。

## 0.1.1 - 2026-05-17

### Added

- 新增首頁說明與首頁視覺圖，降低 header 承載的資訊量。
- 資料查詢新增下拉選單與 sticky 快速切換列。

### Fixed

- 修正資料來源標題與開啟連結內容不一致的問題，並同步更新資料匯出器避免重新產生後回退。
- 修正分享圖卡匯出時內容未置中的問題。
- 修正考績法條文標示，年度試算改標示為第 7 條。

## 0.1.0 - 2026-05-17

### Added

- 新增預設暗黑模式，並提供亮暗模式切換。
- 新增手機版 sticky 橫向分頁列，改善小螢幕切換頁籤體驗。

### Fixed

- 修正薪資試算分頁在切換其他頁籤後仍顯示的問題。
- 修正升等比較結果、年度試算與薪資摘要在手機版的排版溢出。
- 修正部分官方法規與資料來源 URL 導向錯誤頁面的問題。

### Changed

- 強化 RWD：升等比較表單、比較摘要、明細對照與分享圖卡支援窄螢幕。
- 優化暗色主題下的卡片、表格、表單與提示色彩。
